# log.h

## Usage

### Installation

Clone the repository.

```shell
git clone git@github.com:ragibasif/log.h.git
```

Make sure you have `gcc` and/or `clang` and `make` installed.

### Build the project

```shell
make
```

### Run the executable

```shell
make run
```

### Clean build files

```shell
make clean
```

## Dependencies

This project uses:

- C Programming Language
- GNU Make
